HOW TO EDIT YOUR BLC SITE
=========================

You almost never need to touch the HTML. Two files do everything:

1) content.js   →  all your WORDS and which PHOTO goes where
2) the folder   →  drop your actual image files in here

------------------------------------------------------------
TO CHANGE WORDS
------------------------------------------------------------
- Open content.js in any text editor (even Notes / TextEdit).
- Change the text inside the "quotes".
- Keep the quotes and the comma at the end of each line.
- Tricks:
    *asterisks*  = makes that part italic + mauve
    |  (pipe)    = a line break inside a heading
- Save. Done.

------------------------------------------------------------
TO ADD A PHOTO
------------------------------------------------------------
- Put your image file in this folder (same place as content.js).
- In content.js, type its exact filename, e.g.:
      work1Photo: "work-1.jpg",
- Leave a photo field as ""  to keep the plain dark card.

Photo spots available on the homepage:
    heroPhoto    – full-screen background behind the headline
    storyPhoto   – the image beside "The Studio" text
    work1Photo   – first Selected Work card
    work2Photo   – second card
    work3Photo   – third card

------------------------------------------------------------
TIPS
------------------------------------------------------------
- Use web-sized images (under ~500 KB each) so the site stays fast.
- Filenames: no spaces. Use dashes, e.g. "brand-shoot-1.jpg".
- After editing, re-upload the folder to Vercel (or git push).
